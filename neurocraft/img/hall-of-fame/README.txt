HALL OF FAME PHOTOS
===================

Drop your photos in THIS folder to have them show up automatically in the
"hall of fame" window on the site. No code changes needed.

RULES
-----
1. Name the files with plain numbers, starting at 1, with no gaps:
       1.jpg
       2.png
       3.jpeg
       4.webp
       ...

2. Any of these file types work: jpg, jpeg, png, gif, webp

3. The page loads photos in order (1, 2, 3, ...) and STOPS at the first
   number that is missing. So don't skip numbers -- if you have 1, 2, 4
   (missing 3), only 1 and 2 will appear.

4. To remove a photo, delete it and renumber the ones after it so there
   are no gaps (e.g. delete 2.jpg, then rename 3.jpg -> 2.jpg, 4.jpg -> 3.jpg).

That's it -- refresh the page and the new photos appear in the grid and the
prev/next slideshow.
